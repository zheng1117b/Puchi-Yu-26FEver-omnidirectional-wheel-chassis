# News

说明：

- 本目录用于存放消息传输代码。
 本目录包含消息/调试相关的工具模块，主要用于日志打印与遥控数据接收。
作者：扑哧宇  
关键文件：
- `remote_control.c` 进行遥控数据的接收。
 - `my_printf.c` / `my_printf.h`：基于 USART6 + DMA 的轻量级 printf 封装。
 - `remote_control.c` / `remote_control.h`：SBUS 遥控数据接收与解析。

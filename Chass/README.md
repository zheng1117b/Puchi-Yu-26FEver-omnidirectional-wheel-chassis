# Chass

说明：

- 本目录为底盘（Chassis）子系统代码，负责整车移动、轮速控制、电机驱动与底盘状态监测。
- 关键文件：
	- `my_tool.c` / `my_tool.h`：底盘工具函数（运动学、PID 参数/状态等）。
	- `my_drive.c` / `my_drive.h`：底盘运动控制、速度环/电流环接口与 CAN 发送封装。

作者：扑哧宇  
日期：2025-12-23

注意：

- 与主工程集成前请确认 CAN、供电与电机参数（如轮径、减速比）一致。
- PID 参数为经验值，首次上车测试时请在安全条件下逐步调试。 

更新记录：

- 2026-03-22：文档格式化。
